<h1>Blog Express and Typescript</h1>
<h3>Commands</h3>
<p>Run local server</p>
<code>
    
    npm run dev

</code>

<h3>Routes</h3>


    / -> home

    /posts -> All posts

    /posts/:id -> Post by id

    
<h3>View routes</h3>


    /posts -> All posts

    /posts/:id -> Post by id

    /usuarios/:id -> User by id (profile)

    /comentarios/:id -> Comments by id
